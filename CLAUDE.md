# Project Instructions

## Tech Stack
- Next.js 16 with App Router
- React 19, TypeScript
- Prisma with SQLite
- Tailwind CSS + shadcn/ui

## Commands
- `npm run dev` - Start dev server
- `npm run build` - Production build
- `npm run lint` - Run ESLint
- `npm run test` - Run Vitest tests

## Database
- Run `npx prisma db push` to create/update the database
- Run `npm run seed` to seed sample data

## Worktrees
- Use `.worktrees/` directory for isolated branches
- It's already gitignored
